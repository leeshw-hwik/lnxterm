# LnxTerm 개발 워크스루

## 현재 릴리스 기준
- 버전: **v1.9.1**
- 바이너리: `dist/lnxterm`

## 이번 릴리스 핵심 변경
1. **Dark Theme 적용**: `Fusion` 스타일 기반의 VS Code 스타일 다크 테마
2. **자동 명령/통계 개선**:
   - `Last Run` (자동 명령), `Last Detected` (문자열 통계) 시간 표시
   - 타임스탬프 포맷 개선 (밀리초 포함)
3. 다국어(한국어/영어) 및 설정 저장 지원

## 점검 시나리오

### 1) 언어 기본값 확인 (첫 실행)
1. 기존 설정 파일(앱 설정) 제거 또는 신규 환경에서 실행
2. 앱 시작 후 메뉴/라벨이 영어로 보이는지 확인

### 2) 언어 전환 확인
1. 메뉴 `Language -> 한국어` 선택
2. 메뉴/사이드바/검색바/자동 명령 다이얼로그 텍스트가 한국어로 바뀌는지 확인
3. 다시 `Language -> English`로 전환 후 영문 복귀 확인

### 3) 언어 설정 저장 확인
1. 영어/한국어 중 하나를 선택한 상태로 앱 종료
2. 재실행 시 마지막 선택 언어가 유지되는지 확인

### 4) 메뉴 하이라이트 가독성 확인
1. 메뉴바 항목(파일/편집/보기/언어/도움말)에 마우스 오버
2. 드롭다운 항목 선택 시 글자가 배경에 묻히지 않는지 확인
3. VS Code Dark와 유사한 대비(밝은 텍스트 + 진한 배경)인지 확인

### 5) 기존 기능 회귀 확인
1. 시리얼 연결/해제
2. 로그 시작/중지
3. 문자열 통계 카운트
4. 자동 명령 트리거
5. 터미널 검색

## 실행/검증 명령
```bash
python3 -m py_compile i18n.py main_window.py sidebar_widget.py search_widget.py automation_dialog.py styles.py
./build_exe.sh
./dist/lnxterm
```

## 참고
- 빌드 경고(`libxcb-cursor.so.0`, `libtiff.so.5`)가 있어도 산출물이 생성될 수 있습니다.
- 타겟 Linux 환경에서 의존성 설치 상태를 반드시 확인하십시오.
