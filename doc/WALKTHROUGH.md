# LnxTerm 개발 워크스루

## 현재 릴리스 기준
- 버전: **v1.10.0**
- 바이너리: `dist/lnxterm`

## 이번 릴리스 핵심 변경
1. **환경변수 자동 관리**
   - `.env` 자동 생성
   - 기본 `RECONNECT_INTERVAL_MS=1000`
   - 설정 변경 시 즉시 `.env` 저장
2. **자동 명령 안정화**
   - 미연결 시작 차단
   - 정지 시 예약 명령 즉시 중지
   - `sleep(ms)` 지연 명령 지원
3. **문자열 통계 실행 정책 강화**
   - 미연결 시작 차단
4. **매크로 기능 추가**
   - 별도 창에서 등록/실행
   - 최대 1000개, 단축키 10개

## 점검 시나리오

### 1) `.env` 자동 생성 확인
1. `.env` 삭제 후 `./dist/lnxterm` 실행
2. `.env` 생성 및 기본 키 생성 여부 확인

### 2) 포커스 복원 확인
1. 앱 비활성화 후 다시 활성화
2. 명령 입력창 자동 포커스 확인

### 3) 자동 명령 가드/중지 확인
1. 미연결 상태에서 자동 명령 `Start` 시도
2. 연결 후 지연 명령 실행 중 `Stop` 클릭
3. 예약된 후속 명령 즉시 중지 확인

### 4) `sleep(ms)` 실행 확인
1. 자동 명령 세트에 `sleep(500)` 포함
2. 다음 명령 실행이 500ms 지연되는지 확인

### 5) 문자열 통계 가드 확인
1. 미연결 상태에서 통계 `Start` 시도
2. 시작 차단 메시지 확인

### 6) 매크로 확인
1. 매크로 창에서 명령어 등록
2. 번호 셀 클릭 또는 `Ctrl+1`로 즉시 전송 확인

## 실행/검증 명령
```bash
python3 -m py_compile main.py main_window.py sidebar_widget.py macro_dialog.py automation_dialog.py i18n.py
./build_exe.sh
./dist/lnxterm
```

## 참고
- 빌드 경고(`libxcb-cursor.so.0`, `libtiff.so.5`)가 있어도 산출물이 생성될 수 있습니다.
- 타겟 Linux 환경에서 의존성 설치 상태를 반드시 확인하십시오.
