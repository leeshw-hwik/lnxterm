# LnxTerm 구현 계획

ST-Link V3 Mini 기반 임베디드 장치 디버깅/로그 수집용 시리얼 터미널 GUI.

## 현재 릴리스 범위
- 버전: **v1.8.2**
- 목표: 실행환경별 `.env` 적용 안정화, 재연결/로그/통계 지속성 강화, 문서 동기화

## 기술 스택
- Python 3.12 + PyQt6 + pyserial + python-dotenv
- PyInstaller (`./build_exe.sh`)
- GitHub CLI(`gh`) 릴리스 관리

## 핵심 기능 현황

| 기능 | 상태 |
|------|------|
| 시리얼 통신 | 포트 스캔/연결/해제/QThread 수신 |
| 자동 재연결 | 환경변수 기반 주기(`MS`/`SEC`), 기본 3초 |
| 포트 점유 감지 | Lockfile + `fuser`/`lsof` + `TIOCEXCL` |
| 로그 파일 | 세션 고정 파일 유지, 재연결 append 재개 |
| 통계 CSV | 로그 파일 기반 경로 고정, 재연결 후 누적 지속 |
| UI 표시 유지 | 연결만 해제/재연결 시 로그/통계 경로 표시 유지 |
| `.env` 로드 | 실행파일 디렉토리 우선 탐색 + fallback |
| 배포 | PyInstaller 빌드 및 GitHub Release |
