# LnxTerm 개발 워크스루

## 프로젝트 개요
ST-Link V3 Mini 기반 임베디드 장치 로그 수집/터미널 제어용 PyQt GUI 프로그램.

## 현재 릴리스
- 버전: **v1.8.1**
- `./build_exe.sh` 빌드 완료 후 `dist/lnxterm` 생성

## 최근 변경 요약
- 문자열 통계 CSV를 다음 순서로 고정
  - `문자열, 집계 시점, 누적 카운트, 대소문자 구분, 로그`
- 통계 CSV의 `로그` 열은 매칭된 `원본 한 줄 로그`를 저장
- 문서 체계를 GEMINI 규칙에 맞춰 `doc/*.MD`(대문자)로 정리
- GitHub 릴리스 대상 버전은 `v1.8.1`로 갱신

## 주요 기능

### 시리얼/포트
- `Port`, Baudrate, Data, Parity, Stop 설정
- 포트 점유 감지 및 경고( lockfile/fuser/lsof + TIOCEXCL )
- 연결 끊김 시 자동 재연결 지원

### 로그
- 연결 시 자동 로그 생성: `lnxterm_YYYYMMDD_HHMMSS.log`
- 로그 파일 경로를 사이드바에 절대경로로 노출, 경로 복사 버튼 제공

### 문자열 통계
- 최대 10개 문자열 카운터 관리
- 항목별 시작/정지, 누적 카운트, 시작 시각 표시
- 통계 파일 경로 복사 및 항목 전체 초기화 기능
- 로그 라인 단위 카운트 근거 보존을 위해 원문 로그를 CSV에 기록

### 화면 레이아웃
- 사이드바(왼쪽) / 터미널(오른쪽) 분할 조절
- 상단 아이콘으로 연결/해제/클리어 동작

### 배포
- `./build_exe.sh` 실행 시 단일 실행 파일 생성
- GitHub Release에 버전 태그 및 실행 파일 업로드

## 실행/점검
```bash
./build_exe.sh
./dist/lnxterm
```
