# LnxTerm v1.9.1 릴리즈 노트

## 🎨 UI/UX 주요 업데이트
- **다크 테마 적용**: Qt의 `Fusion` 스타일과 커스텀 다크 팔레트를 사용하여 전면적인 다크 모드를 구현했습니다.
  - VS Code와 유사한 현대적인 IDE 미학을 적용했습니다.
  - 가독성을 높이고 눈의 피로를 줄였습니다.
  - 메뉴바, 툴팁, 모든 위젯에 걸쳐 일관된 테마를 제공합니다.

## ✨ 기능 개선
- **자동 명령 및 통계**:
  - 자동 명령 목록에 '마지막 실행(Last Run)' 시간 표시를 추가했습니다.
  - 문자열 통계 아이템에 '마지막 탐지(Last Detected)' 시간 표시를 추가했습니다.
  - 타임스탬프의 정밀도를 개선했습니다 (밀리초 지원).
- **다국어 지원**:
  - 한국어/영어 지원 보강 및 `QSettings`를 통한 설정 유지 기능을 강화했습니다.

## 🐛 버그 수정
- 타임스탬프 포맷 문제 수정 (대괄호 제거, 날짜/시간 정밀도 확보).
- 사이드바 내 사소한 UI 레이아웃 오류를 해결했습니다.
