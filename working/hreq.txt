자동 명령 설정 시 명령어 간격 설정 한계를 24시간 까지 ms로 설정가능 하게 하고, 감지 후 지연 시간도 24시간 까지 ms로 설정가능하도록 변경해줘. 자동 명령 설정 창에서 사전 명령어를 입력하면 몇개가 등록 되었는지 `사전 명령(시작 시 수행)` 라벨 옆에 숫자로 표시 해줘. 사후명령도 동일한 방식으로 명령어 개수 표시해줘.
문자열 통계에 시작 시간 아래에 마지막 검출된 시간을 시작 시간과 동일한 포맷으로 표시해줘.
`자동 명령 정보`영역의 제목을 `자동 명령`으로 한글과 영문 모두 변경해줘.
자동 명령 목록에도 자동 명령 제목 아래에 마지막으로 수행된 날짜와 시간을 표시해줘.
