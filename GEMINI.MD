# LnxTerm 프로젝트 규칙

- 대화 및 문서작성은 한국어를 기본으로 한다.
- 실행해라는 명령 수행은 `reqs/human_requests.txt` 파일의 내용을 확인하고 수행한다.
- 문서 마크다운 파일은 `doc/` 디렉토리 하위에 저장한다.
- Implementation Plan은 `doc/IMPLEMENTATION.MD` 파일에 기록한다.
- AI를 위한 Prompt는 요구사항은 `doc/PROMPT.MD` 파일에 기록한다.
- `.env` 파일은 절대 AI로 전송하지 않는다 (`.antigravityignore` 등록).
- `.env` 파일은 git에 포함하지 않는다 (`.gitignore` 등록).
